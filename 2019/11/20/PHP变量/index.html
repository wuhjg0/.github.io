title: PHP变量

PHP 变量

1.含义与定义形式

使用一个“标记符号”（标识符），代表某个数据

  • 定义形式： $变量名 = 具体的数据;
    ps:变量名区分大小写

    举例： $v1 = 123;
    这里的=,不是数学上的等于，而是赋值的意思
    其本质含义是：将123这个数据放到v1这个变量里去。

    2.变量的命名规则

1. 基本规则

就是给一个变量命名的时候要遵循的的规则。

  1. 构成：字母、数字、下划线；
  2. 开头：字母或下划线；
  3. 变量名不要跟系统中的语法所用单词重复
2. 行业规则

通常，尽量将变量命名为“见名知意”。
尽量用更容易理解的单词（或拼音）去表示一个数据，很多时候可以使用“多个单词（拼音）”。
比如：

  1. ChildAge, YungerAge, ParentHouse, MyParentHouse, MyParentHousePrice 此时，通常有两个行业中的惯例规则：
  2. 骆驼命名法（小驼峰命名法）：第一个单词首字母小写，其余单词首字母大写。 childAge, yungerAge, parentHouse, myParentHouse, myParentHousePrice
  3. 帕斯卡命名法（大驼峰命名法）：所有单词都首字母大写。 ChildAge, YungerAge, ParentHouse, MyParentHouse, MyParentHousePrice

3.变量的4中基本操作

任何一个变量，有且只有4种对变量的”操作”

  1. 赋值–常见操作
    将一个数据（值）放入一个变量中。
    $name = “111”;
    几乎所有变量想要后续正常使用，第一件事就是”赋值”。
    变量初始赋值后，可以后续再去重新赋值–这就是修改了变量的值，比如：
    $name = “666”;
    变量赋值的本质是：将变量名，跟一个数据“建立联系”
  2. 取值–常见操作
    取值 就是从变量中拿到其中存储的数据（值）
    凡是使用到变量，并在该位置上，需要一个数据的时候就会用到取值
  3. 判断变量isset()
    判断一个“变量名”是否里面存储了数据
    判断的结果是：true（表示有） ，false（表示没有）
    使用这个语法来判断：isset(变量名)；

还有一个特殊赋值，赋值后，变量中也没有数据
$name = null;
null是一个特殊的数据，该数据的含义是：没有数据
所以判断 isset($name) 结果是“false”


4. 删除变量unset()
当一个变量中存储了数据，也可以去删除它，语法如下
unset($变量名)
删除变量的本质是：断开变量名跟其关联过的数据之间的联系
删除后 insset()判断结果为false 。

4.变量传值（重点）

1.含义：

是指，将一个变量的值“传递”给另一个变量的方式问题。 形式上，就是一个变量在等号左边，一个变量在等号右边，就表示将右边变量的值传给左边的变量。 有且只有两种传数据的方式：值传递，和引用传递。

2.值传递

就是将右边的变量数据本身，复制一份，然后赋值给左边变量
如下：

$name = $1;
$name2 = $name;

值传递之后，两个变量互不影响，互相独立。

5.预定义变量

在PHP语言内部，有一些变量，是现成的直接可以使用，这就是预定义变量

4.$_GET 变量

代表浏览器表单通过“get”方式提交的所有数据，可以成为“get数据”也可以理解为：
$_GET 变量里面会自动储存 提交到某个文件中的GET数据。
而GET数据，是一个页面以“get”方式请求的时候提交的数据。

5.$_POST

代表浏览器表单通过“post”方式提交的所有数据，称为POST数据
也可以理解为：
$_POST 变量里面会“自动存储” 保存/装载提交到某个文件中的POST数据。
而POST数据，是在一个表单中以“post”方式提交的数据。

6.$_REQUEST

代表浏览器通过“get”方式或“post” 方式提交的数据的合集。
它既能接收到get过来的数据，也能接收到post过来的数据。
通常，一个表单，只能提交一种形式的数据，那么get数据，要么是post数据。

7.$_SERVER变量

它代表任何一次请求中，客户端或服务端的一些“基本信息”或系统信息。
常用的有：

  • PHP_SELF： 表示当前请求的网页地址（不含域名部分）
  • SERVER_NAME： 表示当前请求的服务器名
  • SERVER_ADDR： 表示当前请求的服务器 IP 地址
  • DOCUMENT_ROOT：表示当前请求的网站物理路径（apache 设置站点时那个）
  • REMOTE_ADDR： 表示当前请求的客户端的 IP 地址
  • SCRIPT_NAME： 表示当前网页地址

6.可变变量

含义：变量名本身又是一个“变量”的变量。

实例：

$v1 = 10;
echo $v1;     //输出10 
$str = “v1”;  //这是一个变量，名为 str，值为“v1”（字符串）     
echo $$str;   
//输出 10。这里，“$$str”就是所谓的“可变变量”    
//相当于 echo $v1; 结果就是 10
